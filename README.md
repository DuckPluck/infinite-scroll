### Built using:
1. ReactTS
2. RTK Query
3. React Router

### Project startup:

1. Clone a repository.

   git clone git@github.com:DuckPluck/infinite-scroll.git

2. Enter the project folder.

   cd infinite-scroll

3. Install dependencies.

   yarn

4. Launch project.

   yarn dev
